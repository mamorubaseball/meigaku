# meigakuナビ（リポジトリ/プロジェクト/OSSなどの名前）

明学サークルと学生をマッチングさせるアプリ。
現在追加機能として、学生間で授業の評価が可視化できるよう【みんなの授業】機能を作成中。
また、就活サイトとしても活用してもらうため、就活に関する記事を作成中。

# DEMO

一度wordpressで作成したアプリをDjango環境で再開発。
参考サイト：https://gakuseikoujyou.com/basebaii/

# Features

ユーザー数3000人のアカウントで運用
